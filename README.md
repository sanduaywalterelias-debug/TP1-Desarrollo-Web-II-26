# Finca Dionisia

##Descripción del comercio
Finca Dionisia es una bodega familiar ubicada en santo Tomás, distrito de Tinogasta, provincia de Catamarca.
La finca posee una superficie de 4 hectáreas productivas destinadas al cultivo de vid mediante un sistema tradicional de riego por inundación .
Actualmente se trabajan dos variedades principales de uvas.

-Malbec.
-Criolla.

La producción está destinada a la elaboración de vinos artesanales, alcanzando aproximadamente 1.500 litros anuales. 
El objetivo de este proyecto es desarrollar una aplicación web moderna que permita mejorar la presencia digital de la bodega, difundir sus productos y brindar información sobre su historia, ubicación y proceso de producción.

## ¿ Que es una PWA?
Una PWA ( Progressive App) es una aplicación web que combina características de un sistema web tradicional con funcionalidades similares a las aplicaciones móviles.

Las ventajas para finca Dionisia son:

 1. Acceso desde teléfonos móviles.
 2. Mayor velocidad de carga.
 3. Posibilidad de instalación en dispositivos compatibles.
 4. Mejor experiencia para turistas  y potenciales clientes.
 5. Acceso desde cualquier lugar con conexión a internet. 

## Framework seleccionado
Seleccioné React como framework para el desarrollo del proyecto dado que:

 - Tiene una amplia adopción en la industria.
 - Su arquitectura esta basada en componentes reutilizables.
 - Tiene facilidad de mantenimiento. 
 - Exelente rendimiento para aplicaciones SPA.
 - Es compatible con  futuras funcionabilidades del proyecto.
